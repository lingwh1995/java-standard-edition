第一步：解压并找到核心文件
    解压该 zip 文件后，进入目录（通常在 dist 或根目录下），寻找以下关键文件：
    quaqua.jar: 这是核心外观库。
    libquaqua64.jnilib / libquaqua.dylib: 这是 macOS 原生库（如果你的项目需要调用原生系统转场效果）。

第二步：手动安装到本地 Maven 仓库
    因为官网提供的包没有预设的 Maven 结构，你需要打开终端（Terminal 或 CMD），切换到解压后 quaqua.jar 所在的目录，运行以下命令：
    mvn install:install-file -Dfile=quaqua.jar -DgroupId=ch.randelshofer -DartifactId=quaqua -Dversion=9.1 -Dpackaging=jar

第三步：在 pom.xml 中引用
    安装成功后，你就可以像使用普通插件一样在项目中引用了：
    <dependency>
        <groupId>ch.randelshofer</groupId>
        <artifactId>quaqua</artifactId>
        <version>9.1</version>
    </dependency>

第四步：处理 Native 库 (macOS 专用)
    Quaqua 的强大之处在于它调用了 macOS 的原生 API。为了让它生效，你通常需要在启动 Java 程序时指定 native 库的路径：
    将解压出来的 .jnilib 或 .dylib 文件放在项目的 resources/native 目录下。
    启动参数添加：-Djava.library.path=path/to/your/native/libs
